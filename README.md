# web_app  
Personal Expense Tracker

This project helps to record and manage personal expenses.  
The user can add deposits, expenses, check balance, and see all transactions.  
Also includes a pie chart to visualize expenses by purpose.

---

## Features
- User login and registration  
- Add deposit and expense  
- Check current balance  
- View transaction history  
- Filter by date and amount  
- Expense visualization using pie chart  

---

## Tools and Technologies
- Python (Flask)
- HTML, CSS
- SQLite
- Matplotlib

---

## How to Run
1. Clone this repository  
   ```bash
   git clone https://github.com/krishnareddyyalala/web_app.git
   ```
2. Open the folder
   ```bash
   cd web_app
   ```
3. Install required packages
    ```bash
    pip install flask matplotlib
    ```
4. Run the project
   ```bash
   python app.py
   ```
